# Dentalist
This project is dental app for mobile.
<br>
I coded this project because my big brother wanted a dental app for himself
<br>
I devoloped the project with flutter and sqlite
<br>
## Setup
firstly `cd  mobil_dis_otomasyonu`
<br>
secondly ` pub get `
